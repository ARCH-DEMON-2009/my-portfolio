# Style

A clean, modern, tactile aesthetic where the whole personality is physical depth, every button is a 3D 'pushable' key that lifts on hover and compresses on press. The base is a soft off-white #f6f7fb with crisp near-black ink text (#0f1424) and a muted slate body color (#5b6178). The hero accent is a confident cobalt blue used as a 5-step gradient face, paired with a single warm amber as the secondary/attention color. Inter is the only typeface (weights 400-900), with black (900) tracking-tight display headlines, extrabold (800) section titles, and bold (700) button labels. Surfaces are airy and friendly: a faint cobalt radial dot-grid backdrop behind the hero, two big blurred floating color orbs (cobalt-200/40 + amber-300/30), and white .swatch-card panels with rounded-[22px] corners, a 1px #eceef5 border, and a soft layered drop shadow (0 1px 2px rgba(15,20,36,.04), 0 12px 30px -18px rgba(15,20,36,.18)). The signature is the 3D button itself, three stacked absolutely-positioned layers (a blurred bottom shadow, a static darker 'edge' base, and a raised gradient 'front' that sits translateY(-5px) at rest), animated with a springy transform transition cubic-bezier(.3,.7,.4,1): hover lifts the front to -7px and drops the shadow, :active snaps the front to -2px in 34ms so it physically presses in. Iconography is Iconify ph (phosphor) glyphs. Nothing is flat, the depth, the gradient faces, and the press physics carry the identity.

## Spec

Build a tactile, modern 3D 'pushable' button system on a light base. Set body background #f6f7fb, text color #0f1424 (ink), antialiased. Font: Inter ONLY (Google Fonts, weights 400;500;600;700;800;900) as the sans family; display headlines font-black (900) tracking-[-.03em], section titles font-extrabold (800), button labels font-bold (700) letter-spacing .1px. Define the palette via Tailwind config extend.colors: cobalt 50 #eef2ff, 100 #e0e7ff, 200 #c7d2fe, 300 #a5b4ff, 400 #7c8cff, 500 #4f6bff, 600 #1f4fff, 700 #1539d6, 800 #142fa8, 900 #162a82; amber 300 #ffd089, 400 #ffb84d, 500 #ff9e2c, 600 #f5851f; ink #0f1424. Muted body text is #5b6178. The core is the 3D pushable button .pb: position relative, transparent border-none background, padding 0, cursor pointer, outline-offset 4px, transition filter .25s. It has three stacked children, all border-radius 14px: .pb__shadow (absolute inset-0, background rgba(15,20,36,.18), transform translateY(2px), transition transform .6s cubic-bezier(.3,.7,.4,1)); .pb__edge (absolute inset-0, a darker static base, a left-to-right gradient for rounded-corner shading); and .pb__front (block relative, font-weight 700, transform translateY(-5px), the same .6s springy transition). Hover: .pb filter brightness(1.04), .pb__front transform translateY(-7px) and .pb__shadow translateY(5px) (both fast .25s cubic-bezier(.3,.7,.4,1.5)). Active (pressed): .pb__front translateY(-2px) and .pb__shadow translateY(1px) in .034s, so it snaps in. focus-visible: front box-shadow 0 0 0 3px #f6f7fb, 0 0 0 6px #4f6bff. Surfaces: .swatch-card = background #fff, 1px #eceef5 border, border-radius 22px, box-shadow 0 1px 2px rgba(15,20,36,.04), 0 12px 30px -18px rgba(15,20,36,.18). Backdrop: .dotgrid = radial-gradient(rgba(31,79,255,.10) 1px, transparent 1px) at background-size 22px 22px; .nav-blur = background rgba(246,247,251,.78), backdrop-filter saturate(160%) blur(14px); .hero-orb = radial-gradient(circle at 30% 30%, #3a64ff, #1f4fff 55%, #142fa8 100%). Icons are Iconify ph (phosphor) loaded from code.iconify.design. Keep everything crisp, light, depth-driven and friendly, the physics and the gradient button faces are the whole point. No em-dashes in any copy.
